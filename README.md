

## 💰 Expense Tracker (Java Console Project)

A simple console-based **Expense Tracker application** built using Java.
This project is built to practice **Core Java concepts including classes, objects, collections, and streams**.

---

## 🚀 Features

* ➕ Add expenses with name, amount, and category
* 📋 View all expenses
* 📊 Calculate total spending
* 🔍 Filter expenses by category
* 🧠 Input validation using exception handling
* 🗂️ Category management using Enum
* 💻 Menu-driven console application

---

## 🛠️ Technologies Used

* Java (Core Java)
* Object-Oriented Programming (Classes & Objects)
* Collections (ArrayList)
* Enum
* Exception Handling
* Streams API (basic usage)

---

## 📚 OOP Concepts Used

* ✔ Classes (`Expense`, `Main`, `Category`)
* ✔ Objects (creating and storing Expense objects in ArrayList)
* ✔ Encapsulation (using private fields with getters)
* ✔ Constructors
